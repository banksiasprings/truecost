# TRUE COST

**Know what your car actually costs.**

Australia's most comprehensive vehicle total cost of ownership (TCO) calculator. Built as a Progressive Web App — installable, works offline, mobile-first.

## What it calculates

- Depreciation
- Fuel / Energy (petrol, diesel, EV, PHEV, hybrid)
- EV battery replacement
- Registration (all Australian states)
- Insurance
- Servicing & tyres
- Lost capital (opportunity cost)
- Finance interest

All costs shown as total, per year, and per km. Compare up to 4 vehicles side-by-side.

## Tech stack

Vanilla JS (no framework) · IndexedDB · CSS Variables · Service Worker PWA
